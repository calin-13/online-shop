# Online Shop – Angular Application

A modern single-page web application built using Angular, designed to manage products through a clean and intuitive admin interface.

## Features

- Secure authentication system
- Product management (create, edit, delete, search)
- Responsive admin dashboard
- Real-time updates using reactive programming

## Tech Stack

- Angular (TypeScript)
- RxJS
- Ng-Zorro-Antd UI Library
- LocalStorage

## Description

The application provides a fast and user-friendly interface for managing products in an online shop environment. It supports full CRUD operations, search functionality, and efficient data handling through reactive programming.

## Notes

This project focuses on frontend development and simulates real-world admin dashboard functionality.
